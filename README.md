# 🍽️ Digital Menu - Tio Naná

A responsive digital menu built for a local restaurant, designed to improve customer experience and simplify ordering via WhatsApp.

---

## 🚀 Features

* 🔎 Filter items by category (All, Meals, Desserts, Drinks)
* 📱 Direct order via WhatsApp
* 🎨 Clean and modern UI
* ⚡ Fast and lightweight

---

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript

## 💡 About the Project

This project was created to help a local business improve customer interaction and increase sales through a simple and intuitive digital menu.

It focuses on usability, speed, and accessibility.
